Study-Planner📖📚
A modern and interactive web application that helps students plan, organize, and track their study tasks efficiently. It includes features like task management, time-based planning, progress tracking, and smart reminders with notifications. Built using HTML, CSS, and JavaScript with a responsive and attractive user interface..

🚀 Features
✅ Add, edit, and delete tasks
📅 Set due dates and reminders
🔥 Priority levels (High, Medium, Low)
📊 Dashboard to track tasks
⏰ Reminder alerts with popup notification
🎯 Organized task management

🛠️ Technologies Used
Frontend: HTML, CSS, JavaScript
Storage: Local Storage (for saving tasks)

✨ How It Works
Users can add tasks with title, priority, and due date
Tasks are saved in local storage
Dashboard shows task status
Reminder page gives popup notification
